SandPond, 2019

I made SandPond, a 3D voxel sandbox. I also made SpaceTode, a spatial programming language for creating elements in the simulation by drawing before-and-after pictures. I documented my progress in a series of videos known as the SandPond Saga. SandPond is now finished, but the series continues to this day!

CellPond, 2021

I created CellPond based on everything I learned from SandPond. Unlike SandPond, you can code it entirely by dragging-and-dropping shapes around in its visual programming interface, which was its main design constraint.

Sandspiel Studio, 2022

I created Sandspiel Studio with Max Bittker. It's a simulation game that lets users create new custom elements by dragging-and-dropping blocks in its visual editor. There's a gallery for sharing, exploring and remixing creations.
