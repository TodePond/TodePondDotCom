# Doom is here

Open source software

What a lovely idea that is! If we give our— the software we make an open source license, then that stops people from using it in ways that we don't want. It makes sure that all development on top of our software gets fed back into the community / an ecosystem grows / space is made for collaboration / evil wrongdoing with our software becomes prohibited or i suppose not— evil wrongdoing with our software becomes at the very least visible. 

Except, in some cases, it doesn't seem to work, does it? Despite some claims that AGPL might prohibit companies from training on your software, even if that were proved true, I doubt it'll stop them. Vibe coding on top of AGPL-licensed code might also be against the rules, but I don't think that would stop people / it doesn't seem to stop people. 

With vibe coding, the idea is this: With vibe coding, you're no longer writing the code, right? You're writing the chat messages that you write, right? So the code is no longer the "source" of the software: It's more of a compiled artifact that you ship. The real source code is the chat messages you wrote, as well as the gigantic large language model that you used, which you now also have to open source. 

Some large language models are open weights, but weights are also [just] the post-training numbers that gets compiled from the source, which is the entirety of the internet and all text written by people ever. So, to be in compliance with AGPL when shipping your vibe coded fork, you need to open source the entirety of the human corpus of— you need to open source *everything*— sure, here's my source: everything— forget everything you— here it is: wikipedia and also every book, including all the different language wikis including French and Simple English.

I barely understand it anyway though so who knows what I know, but I fear that it's a technicality that, even if it's true, won't make a practical difference, maybe I'm a doomer after all, but that's not a prediction, I'm not predicting doom will come next year or the next six months no: Doom is here today, right now, doom is here: It's not working.
