# Author's privilege

Only I can make the art I make. no one else can make it. they could try, but they would fail: It has to be me.

This is a belief I adopted three years ago from an artist friend of mine. They told me about the importance of "it has to be me" as a mantra and I inferred more things from it: Inevitability. Decision. Necessity. Specialness.
