# winning winning winning 

winning is good! 

<br>

<br>

if you win it means you did what you wanted to do. it means you found success. i think: by focusing on winning: by focusing on how to win, you end up finding the unexpected and creative solutions. 

this is how cellpond ended up [how it is](https://youtu.be/eQgxFuw8f1U). it's because i had a criteria[^no] for what winning was. it meant that i wasn't afraid of getting weird because all i cared about was winning: results, not what anyone expected of me. i wasn't trying to be [good](https://www.pastagang.cc/blog/bad-code-only/): i was trying to *win*.

i explained this in one of the answers i gave to one of the questions in [that talk](https://youtu.be/eQgxFuw8f1U) but i don't think anyone heard it. 

and no one heard it when i wrote about [winning](https://www.todepond.com/sky/how-to-win/) last time.

and no one will hear it this time either.
